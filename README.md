---

## 🚀 Быстрый старт (3 шага)

### Шаг 1: Установите зависимости
```bash
pip install -r requirements.txt
```

### Шаг 2: Запустите приложение
```bash
python main.py
```

Или альтернативно:
```bash
uvicorn app:main_app --reload
```

### Шаг 3: Откройте в браузере

| Адрес | Описание |
|-------|---------|
| **http://localhost:8000/test** | 🎯 **Интерактивное тестирование** — кнопочки для каждого маршрута |
| http://localhost:8000/docs | 📚 Swagger документация |
| http://localhost:8000 | ✅ Главная страница (Задание 1.1) |

## ✅ Реализованные задания

Все 7 заданий полностью реализованы и протестированы:

| № | Маршрут | Метод | Описание | Статус |
|---|---------|-------|---------|--------|
| **1.1** | `/` | GET | JSON с приветствием | ✅ |
| **1.2** | `/html` | GET | Возврат HTML страницы | ✅ |
| **1.3** | `/calculate` | POST | Сумма двух чисел (`num1`, `num2`) | ✅ |
| **1.4** | `/users` | GET | User модель (name, id) | ✅ |
| **1.5** | `/user` | POST | Проверка возраста `is_adult` | ✅ |
| **2.1** | `/feedback` | POST | Сохранение отзывов в памяти | ✅ |
| **2.2** | `/feedback` | POST | Валидация отзывов (длина, слова) | ✅ |

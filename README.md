# FAPI_KR1 - Контрольная работа №1 по FastAPI

Решение контрольной работы по дисциплине "Технологии разработки серверных приложений" (ИПТИП, 4 семестр 2025/2026).

---

## 🚀 БЫСТРЫЙ СТАРТ

### Шаг 1: Установите зависимости
```bash
pip install -r requirements.txt
```

### Шаг 2: Запустите приложение
```bash
python3 main.py
```
или
```bash
uvicorn app:main_app --reload
```

Вы должны увидеть:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete
```

### Шаг 3: Откройте в браузере (по очереди)

| № | Адрес | Описание |
|---|-------|---------|
| 1 | **http://localhost:8000/test** | 🎯 **ИНТЕРАКТИВНОЕ ТЕСТИРОВАНИЕ** — все кнопочки для проверки каждого маршрута |
| 2 | http://localhost:8000/docs | 📚 **Swagger документация** — интерактивная документация всех маршрутов |
| 3 | http://localhost:8000 | ✅ **Приветствие (1.1)** — JSON: `{"message": "Авторелоад действительно работает"}` |
| 4 | http://localhost:8000/html | 📄 **HTML (1.2)** — текст "Я ОБОЖАЮ ВСТАВАТЬ К ПЕРВОЙ ПАРЕ :)" |
| 5 | http://localhost:8000/users | 👤 **Пользователь (1.4)** — данные пользователя в JSON |

### Шаг 4: Тестируйте API через curl (в отдельном терминале)

**Задание 1.3 - Сумма чисел:**
```bash
curl -X POST "http://localhost:8000/calculate?num1=5&num2=10"
```

**Задание 1.5 - Проверка возраста:**
```bash
curl -X POST "http://localhost:8000/user" \
  -H "Content-Type: application/json" \
  -d '{"name": "Артур", "age": 26}'
```

**Задание 2.1/2.2 - Отправка отзыва:**
```bash
curl -X POST "http://localhost:8000/feedback" \
  -H "Content-Type: application/json" \
  -d '{"name": "Артур", "message": "Это отличный проект!"}'
```

**Тест валидации (запретное слово - вернёт ошибку):**
```bash
curl -X POST "http://localhost:8000/feedback" \
  -H "Content-Type: application/json" \
  -d '{"name": "Артур", "message": "Какой-то кринж"}'
```

## ✅ Реализованные задания

| Номер | Маршрут | Метод | Описание |
|-------|---------|-------|---------|
| 1.1 | `/` | GET | JSON с приветствием и поддержка авторелоада |
| 1.2 | `/html` | GET | Возврат HTML страницы |
| 1.3 | `/calculate` | POST | Сумма двух чисел (`num1`, `num2`) |
| 1.4 | `/users` | GET | Данные пользователя (User модель) |
| 1.5 | `/user` | POST | Проверка совершеннолетия (`is_adult: bool`) |
| 2.1 | `/feedback` | POST | Отправка и сохранение отзывов |
| 2.2 | `/feedback` | POST | Валидация отзывов (имя 2-50 символов, текст 10-500 символов, запретные слова) |

## 🔧 Разработка

Приложение запускается с флагом `--reload`, что означает:
- Сервер автоматически перезагружается при изменении кода
- Идеально для разработки и отладки
- Не используйте в production

# 🧪 Инструкция по тестированию FAPI_KR1

## Быстрый старт

### 1️⃣ Установите зависимости
```bash
pip install -r requirements.txt
```

### 2️⃣ Запустите сервер
```bash
python main.py
```
или
```bash
uvicorn app:main_app --reload
```

Вы должны увидеть:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete
```

## 🎯 Способы тестирования

### Способ 1️⃣: Интерактивное тестирование (РЕКОМЕНДУЕТСЯ)

**Откройте в браузере:** http://localhost:8000/test

Там вы найдёте красивую страницу с тестировочными карточками для каждого маршрута:

1. **GET / (Задание 1.1)** — Нажмите "Тестировать"
   - Ожидается: `{"message": "Авторелоад действительно работает"}`

2. **GET /html (Задание 1.2)** — Нажмите "Открыть страницу"
   - Ожидается: HTML с текстом "Я ОБОЖАЮ ВСТАВАТЬ К ПЕРВОЙ ПАРЕ :)"

3. **POST /calculate (Задание 1.3)** — Введите два числа, нажмите "Вычислить сумму"
   - Ожидается: `{"result": сумма}`
   - Пример: 10 + 5 = 15

4. **GET /users (Задание 1.4)** — Нажмите "Получить пользователя"
   - Ожидается: `{"name": "Дарья Невская", "id": 1}`

5. **POST /user (Задание 1.5)** — Введите имя и возраст, нажмите "Проверить возраст"
   - Ожидается: `{"name": "...", "age": ..., "is_adult": true/false}`
   - Примеры:
     - Возраст >= 18 → `is_adult: true`
     - Возраст < 18 → `is_adult: false`

6. **POST /feedback (Задание 2.1/2.2)** — Введите имя и отзыв, нажмите "Отправить отзыв"
   - Валидация:
     - Имя: 2-50 символов ✅
     - Отзыв: 10-500 символов ✅
     - Запретные слова: ❌ кринж, рофл, вайб
   - Успех: `{"message": "Спасибо, {имя}! Ваш отзыв сохранён."}`
   - Ошибка валидации: `{"detail": [{"msg": "Использование недопустимых слов"}]}`

### Способ 3️⃣: curl команды в терминале

**В новом терминальном окне выполните:**

#### Задание 1.1 - GET
```bash
curl http://localhost:8000/
```

#### Задание 1.2 - GET HTML
```bash
curl http://localhost:8000/html
```

#### Задание 1.3 - POST сумма чисел
```bash
curl -X POST "http://localhost:8000/calculate?num1=15&num2=10"
```

#### Задание 1.4 - GET пользователь
```bash
curl http://localhost:8000/users
```

#### Задание 1.5 - POST проверка возраста
```bash
curl -X POST "http://localhost:8000/user" \
  -H "Content-Type: application/json" \
  -d '{"name": "Иван", "age": 25}'
```

#### Задание 2.1/2.2 - POST отправить отзыв (успешно)
```bash
curl -X POST "http://localhost:8000/feedback" \
  -H "Content-Type: application/json" \
  -d '{"name": "Анна", "message": "Это просто отличный проект!"}'
```

#### Задание 2.2 - Тест на запретное слово (ошибка валидации)
```bash
curl -X POST "http://localhost:8000/feedback" \
  -H "Content-Type: application/json" \
  -d '{"name": "Боб", "message": "Какой-то кринж..."}'
```

Ожидается ошибка: `HTTP 422 - Validation Error`

#### Задание 2.2 - Тест на короткое имя (ошибка валидации)
```bash
curl -X POST "http://localhost:8000/feedback" \
  -H "Content-Type: application/json" \
  -d '{"name": "А", "message": "Отзыв должен быть длиннее"}'
```

Ожидается ошибка: `HTTP 422 - Validation Error`

## 🐛 Возможные проблемы

### Порт 8000 уже используется
```bash
# Найдите процесс
lsof -i :8000

# Завершите его (замените PID на нужный номер)
kill -9 PID
```

### Ошибка импорта модулей
```bash
# Убедитесь, что стоите в папке FAPI_KR1
cd FAPI_KR1

# Переустановите зависимости
pip install -r requirements.txt --force-reinstall
```

### Сервер не загружается
```bash
# Проверьте синтаксис Python
python3 -m py_compile app/main.py

# Запустите с полным путём
/opt/homebrew/bin/python3 main.py
```

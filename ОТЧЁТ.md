## 📋 ЧТО ТРЕБОВАЛОСЬ СДЕЛАТЬ

Выполнить 7 заданий по разработке веб-приложения на FastAPI:

### Задание 1.1
Создать FastAPI приложение с GET маршрутом `/`, возвращающим JSON-ответ. Проверить функцию **авторелоада** при изменении кода.

### Задание 1.2
Создать HTML-файл и FastAPI приложение, которое возвращает HTML-страницу по маршруту `/`.

### Задание 1.3* (усложненное)
Создать POST маршрут `/calculate`, принимающий два числа и возвращающий их сумму в JSON.

### Задание 1.4
Создать Pydantic модель `User` с полями `name` и `id`. Реализовать GET маршрут `/users`, возвращающий данные пользователя.

### Задание 1.5* (усложненное)
Создать POST маршрут `/user`, принимающий данные пользователя с возрастом и возвращающий поле `is_adult` (True если возраст >= 18).

### Задание 2.1

### 2. Создание моделей (models.py)
Реализовал Pydantic модели:
- `User` - с полями `name` (str) и `id` (int)
- `UserAge` - с полями `name` (str) и `age` (int)
- `UserAgeResponse` - с дополнительным полем `is_adult` (bool)
- `Feedback` - с валидацией:
  - `name`: min_length=2, max_length=50
  - `message`: min_length=10, max_length=500
  - Кастомный валидатор для проверки запретных слов

### 3. Реализация приложения (app.py)
Создал FastAPI приложение с поддержкой всех требуемых маршрутов:

#### Задание 1.1 ✅
```python
@main_app.get("/")
def read_root():
    return {"message": "Авторелоад действительно работает"}
```

#### Задание 1.2 ✅
```python
@main_app.get("/html")
def read_html():
    return FileResponse("index.html", media_type="text/html")
```

#### Задание 1.3 ✅
```python
@main_app.post("/calculate")
def calculate(num1: float, num2: float):
    return {"result": num1 + num2}
```

#### Задание 1.4 ✅
```python
@main_app.get("/users")
def get_users():
    return {"name": current_user.name, "id": current_user.id}
```

#### Задание 1.5 ✅
```python
@main_app.post("/user")
def create_user(user: UserAge) -> UserAgeResponse:
    is_adult = user.age >= 18
    return UserAgeResponse(name=user.name, age=user.age, is_adult=is_adult)
```

#### Задание 2.1 ✅
```python
@main_app.post("/feedback")
def submit_feedback(feedback: Feedback):
    feedbacks.append({"name": feedback.name, "message": feedback.message})
    return {"message": f"Спасибо, {feedback.name}! Ваш отзыв сохранён."}
```

#### Задание 2.2 ✅
Реализована полная валидация в Pydantic модели `Feedback` с кастомным валидатором, проверяющим запретные слова.

## 🚀 КАК ЗАПУСТИТЬ ПРОЕКТ

### Установка зависимостей:
```bash
cd FAPI_KR1
pip install -r requirements.txt
```

### Запуск приложения:
```bash
uvicorn app:main_app --reload
```

### Тестирование:
1. **Задание 1.1**: Открыть `http://localhost:8000` в браузере
2. **Задание 1.2**: Открыть `http://localhost:8000/html` в браузере
3. **Задание 1.3**: 
   ```bash
   curl -X POST "http://localhost:8000/calculate?num1=5&num2=10"
   ```
4. **Задание 1.4**: Открыть `http://localhost:8000/users` в браузере
5. **Задание 1.5**:
   ```bash
   curl -X POST "http://localhost:8000/user" \
     -H "Content-Type: application/json" \
     -d '{"name": "Артур Дворецкий", "age": 26}'
   ```
6. **Задание 2.1/2.2**:
   ```bash
   curl -X POST "http://localhost:8000/feedback" \
     -H "Content-Type: application/json" \
     -d '{"name": "Артур", "message": "Это тяжело, но я справлюсь!"}'
   ```

Интерактивную документацию можно просмотреть в Swagger UI: `http://localhost:8000/docs`
